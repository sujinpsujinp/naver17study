# 루트계정
$ 일반 계정

0.  clear : 화면 지우기 
0.  history : 사용했던 모든 명령어 보여주기

1. man ls : ls사용법 확인  >> q 누르면 빠져나감

2. pwd  : 현재위치 확인

3. ls : 목록보기
-a : 숨김파일이나 폴더까지 모두 출력 (젤 중요) 
-l : 목록형식으로 세로 방향으로 출력 (젤 중요) al, hl 처럼 붙여서 많이 사용함
-h : 크기를 바이트에서 M단위로 변경해서 출력
-S : 크기의 내림차순으로 출력
ls -폴더명 : 폴더 안에 목록 보기

4. cd : 디렉토리 변경
- cd ~ : 꼭대기(루트)
- cd /c/naever1210 : c드라이브에 naver1210폴더로 이동
- cd .. : 상위 폴더로 이동
- cd - : 직전 디렉토리로 이동

5. mkdir : 디렉토리 생성
-p : 하위 디렉토리까지 한번에 생성
- mkdir hello : hello 디렉토리 생성
- mkdir -p bitcamp/member/study : 하위디렉토리까지 한번에 생성

6. rmdir : 비어있는 디렉토리 삭제(비어있지 않는 경우 오류 발생)

7. rm : 파일이나 디렉토리 삭제 시 사용 (rm 파일명 : 비어있지 않은 경우 오류 발생)
- -r : 디렉토리 삭제 시 하위 디렉토리도 같이 삭제  (-rf로 붙여서 많이 사용함)
- -f : 삭제물음없이 강제 삭제 

8. cat : 파일 안의 내용 확인
- -n : 번호 출력

9. cp : 복사
- -r : 하위디렉토리까지 모두 복사
- -b : 복사될 파일이름이 이미 존재할때 백업파일을 생성
- -f  : 복사될 파일이름이 이미 존재할때 강제로 덮어쓰기
- -i : 복사될 파일이름이 이미 존재할때 덮어쓰기 여부를 물음

8. vi 에디터 : 문서 편집
- :w : 저장
- :q or :q!  : 저장 안하고 종료
- :wq : 저장 후 종료
- ! or a : 편집 모드
- esc 키 : 명령 모드
- dd : (명령모드) 한줄 삭제(잘라내기)
- p : (명령모드) 붙여넣기
- :set nu : 라인번호 (취소 : set nonu)
- /문자열 : (명령모드)해당 문자열로 이동
- 숫자 : (명령모드)해당라인으로 이동
- 줄의 맨앞 : 0(숫자 0)
- 줄의 맨 뒤 : $
